React Native Quiz App

The goal of this assignment was to create a simple React Native quiz app that accepts a list of questions and answers and renders them to the screen. This app supports React Native Navigation, and contains a Question component and a Summary component. This app also needed to support three types of answers: True/False, Multiple Choice, and Multiple Answer. The Summary screen displays the users total score and displays all questions and whether the user chose the correct or incorrect answer.

UI Design Projects

Study Smarter

The goal of this assignment was to design a mid-fidelity mobile application that allows users to study flashcards. This application would allow users to create, delete and edit flashcards and view them in a random order. Each flashcard has a prompt on one side and answer on the other. My prototype also includes a search feature that will allow the user to look for existing,public flashcards decks. They also have the option to save and favorite cards that they enjoy.

Task Track

The goal of this assignment was to design a mid-fidelity TODO app that will allow users to create and manage tasks, set deadlines, and mark tasks as completed. My prototype consists of a signup/login page, profile, and a main page that allows the user to navigate to their tasks, upcoming tasks, completed tasks, and view their calendar. The task page will allow users to add new tasks to their list, check off completed tasks, edit their task name and description, and delete tasks.
