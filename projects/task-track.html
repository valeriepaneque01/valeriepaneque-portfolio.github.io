Task Track

Visual Design / UI+UX / Website Development

Mid-fidelity TODO application

The goal of this assignment was to design a mid-fidelity TODO app that will allow users to create and manage tasks, set deadlines, and mark tasks as completed. My prototype consists of a signup/login page, profile, and a main page that allows the user to navigate to their tasks, upcoming tasks, completed tasks, and view their calendar. The task page will allow users to add new tasks to their list, check off completed tasks, edit their task name and description, and delete tasks. Please view my Figma prototype below.
